## Description
A simple Python program to perform matrix addition without using external libraries.  
This project demonstrates basic matrix operations using nested lists and loops.

## Features
- Adds two 3x3 matrices
- Displays input matrices and the result
- No external libraries required (pure Python)

## How to Run
1. Clone or download the repository.
2. Open terminal in the project folder.
3. Run the Python file:

```bash
python matrix_add.py
The program will display the input matrices and their sum.
